Zyappy POS — Catalog Intelligence / Items Module 
Final Developer Handoff Document (Complete Version) 
Module: 
M1 – Catalog Intelligence / Menu Engine / Item Management 
Purpose 
This module powers Zyappy’s complete menu architecture across: 
•POS 
•Online Ordering 
•Kiosk 
•KDS 
•Uber / Delivery Channels 
•Future marketplace integrations 
This document defines complete frontend behavior, backend structure, APIs, runtime rules, store 
permissions, publishing lifecycle, conflict resolution, and operational recovery requirements. 
This follows Zyappy’s pizza-first architecture where menu logic is centralized and reused across 
channels.
